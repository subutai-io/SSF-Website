Subutai™ P2P Cloud Platform


The world’s first intelligent Peer-to-Peer (P2P) Cloud Computing platform



Subutai™ originated as a multi-million dollar advanced defense research project to create a new Peer-to-Peer (P2P) Cloud Computing platform that intelligently reconfigures itself dynamically and securely across global resources.

Today Subutai disrupts, democratizes, and commoditizes Cloud and Internet of Things (IoT) services through the following products:

Subutai™ Blockchain Router — power-efficient (18-60w) “green” advanced broadband Cloud router and open hardware IoT gateway.

- Residential Edition — serves as a secure IoT “Swiss Army Knife”, plug-and-play cryptocurrency wallet, and crypto mining device at a hashrate of 232 MH/s. Advanced configuration features bespoke Subutai-tuned FPGA chips to mine multiple cryptocurrencies; provides Internet subscribers ~$115/mo. passive income (at current depressed rates) with no technical skills needed. Ethical mining for the masses.

- Industrial Edition — used in mission-critical Industrial IoT/Industry 4.0 applications, with higher grade temperature-resistant chips, and advanced security via self-adaptive, real-time intrusion detection/prevention systems. Currently in production and being tested at the world's largest producer of renewable energy, among other IIoT and smart city programs.

Subutai™ PeerOS — Open Source, container-based P2P Cloud and IoT software and firmware that allows users to easily create Virtual Private Clouds. Mature v7.0, stable; commercialized for mass consumption over a compute resource and device commodity exchange on the blockchain.Keep your private data private. Download and explore.

Subutai™ Bazaar — a global reputation-based marketplace to buy/sell/exchange computing resources, cloud applications, and idle computer resources; “the Airbnb of computing resources”. Contains software products such as the popular Subutai Blockchain-in-a-Box and IoT Sandbox blueprints. Mature v7.0, stable; used daily by 850+ commercial and individual users in 85 countries. Get started today.

KHAN™ — an Ethereum Blockchain-based reserve currency token that is the default and ubiquitous currency and staking token of the Subutai platform (SEC-complaint code built into the KHAN contract). Key part of dual token structure: GoodWill (Subutai Sidechain smart token for low latency transactions, used to buy/sell Subutai Bazaar resources and products); and KHAN (cryptocurrency used on the Subutai Bazaar for provider and consumer reputation “staking” via SLA escrows). Launching early 2019.

Subutai is proven, solves real-world problems, and is in use today in countless applications by an rom individual enthusiasts to multinational concerns powering mission-critical Systems. Join us!
