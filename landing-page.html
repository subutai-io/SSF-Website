Cloud computing is expensive and dominated by a few major conglomerates with no interest in user privacy.

The Subutai platform leverages peer-to-peer cloud computing, energy-efficient cryptocurrency mining, and blockchain technology to provide users with a holistic solution to this problem. We’re making it easy to utilize untapped computing resources.

Subutai PeerOS

Open Source, container-based P2P Cloud and IoT software that allows users to buy, share, sell, and rent computing power

Subutai Bazaar

Global marketplace for trading computing resources and applications. The Bazaar is powered by GoodWill, a lightweight digital asset used to buy and sell computing resources.

Subutai Blockchain Router v2.0

An energy-efficient broadband router that also serves as a plug-and-play device to mine and store cryptocurrency.

Subutai Blueprints

Templates that reduce system administration overheads and simplifies the process of application deployment. Deploy anything from Blockchain-in-a-Box to a Wordpress site or a Minecraft server with a few easy clicks.
